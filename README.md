# Atom.io http-requestor Package

A simple Atom.io http-post to test RESTful API

This is a Work-In-Process

# Install:
```
apm install http-post
```

# Usage:
```
  module.export = {
    hostname: [Host IP or DNS],
    path: [Path for Resrouce],
    method: [GET | POST]
    body: { - Optional - }
  }
```
